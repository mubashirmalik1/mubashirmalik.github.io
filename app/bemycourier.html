Bemycourier

Project information

  • Category: Mobile App

The app is used for delivery purpose between cities and countries. It has different features, such as sign in with google, facebook and with phone number, Packages list,Upload image and enter details of package you need to get shipped or shopped, Google map with Google Places API integration helps you choose the origin and destination for your package, Nearby list of users trips and their location using Google's Static Map API,Notification, and user management.

Designed by BootstrapMade
